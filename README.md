## Building

The following libraries must be installed to build with cmake
- glfw3 3.3 REQUIRED
- OpenGL REQUIRED
- kissfft REQUIRED
- glm REQUIRED
- portaudio portaudio REQUIRED
- sndfile sndfile REQUIRED
